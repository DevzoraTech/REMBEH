# REMBEH Product Roadmap

## Foundation

- Workspace registration with email OTP.
- Branch creation with address, GPS, phone, and working hours.
- Invitation flow for branch managers and staff.
- Role, permission, and role group system.
- Module registry and module-based pricing.
- Web dashboard for operations leaders.
- Mobile dashboard that changes by permission.

## MVP Modules

- Authentication
- Workspace
- Branches
- Employees
- Roles and permissions
- Customers
- Loans
- Collections
- Cashiers
- Reports
- Notifications
- Audit logs
- Files
- Settings
